# Introduction

Welcome to **KRAG Translate** - a simple yet powerful translator package using Google Translate. Convert your entire website to your desired language with multiple output formats.

## Features

KRAG Translate provides three different implementation styles to suit your needs:

1. **Dropdown Translate** - A sleek dropdown selector with flag icons
2. **Flag-list Translate** - A vertical list of languages with flags
3. **Text Translate** - A simple text-based language selector

## Quick Start

To get started with KRAG Translate, you'll need to:

1. Include the Google Translate script
2. Add jQuery (required for dropdown functionality)
3. Include the KRAG Translate CSS and JavaScript files
4. Add the HTML markup for your preferred style

Check out the [Implementation Styles](./category/implementation-styles) section to see detailed examples of each style.

## Why KRAG Translate?

- **Easy Integration** - Simple HTML and JavaScript setup
- **Multiple Styles** - Choose the UI that fits your design
- **Customizable** - Easy to modify and extend
- **Google Powered** - Uses Google Translate's reliable translation engine
- **Cookie Support** - Track and respond to language changes

## Browser Compatibility

KRAG Translate works with all modern browsers that support Google Translate.
