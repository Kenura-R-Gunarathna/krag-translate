# KRAG Translate - Live Examples

This folder contains live HTML examples demonstrating the three implementation styles of KRAG Translate.

## Examples

- **[dropdown.html](./drop-down.html)** - Dropdown style implementation
- **[flag-list.html](./flag-list.html)** - Flag-list style implementation  
- **[text.html](./text.html)** - Text style implementation

## How to Use

1. Open any HTML file in your browser to see the live demo
2. Click on different language options to see the translation in action
3. View the source code to understand the implementation

## Documentation

For complete documentation, visit: https://kenura-r-gunarathna.github.io/krag-translate/

## Assets

The examples use assets from the `assets/` folder in the parent directory:
- CSS: `../assets/css/app.css`
- JavaScript: `../assets/js/app.js`
- Images: `../assets/img/`
